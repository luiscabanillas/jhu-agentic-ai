# AI Inventory

Small Week 1 Python project for a hardware store billing workflow.

## Features

- Select a tool from the inventory dropdown
- Add tools to a shopping cart
- Remove tools from the cart
- Display current cart contents
- Calculate the total bill

## Files

- `app.py`: Gradio user interface
- `inventory.py`: inventory data and cart logic
- `requirements.txt`: dependency list

## Run

```bash
cd "Week 1/ai-inventory"
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```
